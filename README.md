
## Installation

### Step 1
```
git clone git@github.com:pentest-khoa-02/Group1.git
```

### Step 2
Running
```
docker-compose up --build
```

### Step 3
Go to: http:localhost:3000/fakedata

### Step 4
Go to: http:localhost:3000/settings
